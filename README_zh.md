## 📦 安装方法

### 方式一：手动安装（通用）

```bash
# 创建技能目录（如果不存在）
mkdir -p ~/.claude/skills/

# 克隆或复制技能
git clone https://github.com/YOUR_USERNAME/dad-skill ~/.claude/skills/dad-skill
```

### 方式二：单文件安装（快速）

将 `SKILL.md` 复制到项目的 `.claude/skills/` 或 `~/.claude/skills/` 目录下。

### 方式三：OpenCode

OpenCode 会自动从 `.claude/skills/` 加载所有技能——只需将文件夹放入即可。

## 🎯 使用方法

直接和你的 AI 对话，使用触发词即可：

> "Dad，我需要帮忙..."
> "爸爸，帮我..."
> "Help me dad, I'm stuck on..."

技能会在检测到触发词时自动激活。

### 触发词

`Dad`, `daddy`, `爸爸`, `父亲`, `help me dad`, `I need you dad`, `son`, `儿子`, `孩子`, `my machine`, `skills`
