## 📦 Installation

### Option 1: Manual (any platform)

```bash
# Create skills directory if it doesn't exist
mkdir -p ~/.claude/skills/

# Clone or copy the skill
git clone https://github.com/YOUR_USERNAME/dad-skill ~/.claude/skills/dad-skill
```

### Option 2: Single file (quick)

Copy `SKILL.md` into your project's `.claude/skills/` or `~/.claude/skills/` directory.

### Option 3: OpenCode

OpenCode automatically loads all skills from `.claude/skills/` — just drop the folder in place.

## 🎯 How to Use

Simply talk to your AI and mention trigger words like:

> "Dad, I need help with..."
> "爸爸，帮我..."
> "Help me dad, I'm stuck on..."

The skill auto-activates on these triggers.

### Trigger Words
`Dad`, `daddy`, `爸爸`, `父亲`, `help me dad`, `I need you dad`, `son`, `儿子`, `孩子`, `my machine`, `skills`
